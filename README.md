
# General Knowledge Quiz (Python)

This is a simple interactive quiz program built using Python. It asks multiple-choice or short-answer questions and calculates the final score.

## Features
- Multiple quiz questions
- Score tracking system
- Simple user interaction via terminal

## Skills Used
- Python basics
- Conditional statements (if-else)
- User input handling
- String comparison

## How to Run
1. Open terminal
2. Run: python gk_quiz.py# gk-quiz-python
